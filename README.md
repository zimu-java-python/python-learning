# Python & Java Learning for Japan IT Job
## 日本IT就職のためのPython & Java学習

本仓库用于记录我学习 Python 和 Java 的全过程，目标是通过扎实的技术基础和持续的学习痕迹，成功入职日本 IT 企业。

This repository tracks my learning journey of Python and Java, aiming to secure a position in a Japanese IT company with solid technical foundation and consistent learning traces.

## 📁 仓库结构 (Repository Structure)

- `00_java_core/`: Java 核心学习目录（主要学习重点）
  - `01_hello_java.java`: Java 入门程序，演示基本结构和 main 方法。
- `01_python_basic/`: Python 基础语法练习
  - `first_python_lesson.py`: Python 入门练习，包含变量、计算和条件判断。
- `02_python_cases/`: Python 综合案例和实战项目
  - `03-求钱包余额练习.py`: 函数综合案例，模拟钱包余额查询。

## 🎯 学习规划 (Study Plan)

- **主线**: 主攻 Java，从基础语法到面向对象，再到 Web 开发。
- **辅助**: 跟进 Python，巩固基础，并用其辅助 Java 学习。
- **目标**: 2027年4月入学关西大原语言学校，毕业后获得日本 IT 工作签证。
